# Link-TV

#### 介绍
一套包含了推流端，服务端，web前端的适用于游戏直播的系统。采用了前后端分离的方式进行开发。  
Linkme：一个专门运行于windows下的推流软件。  
LinkServer：为直播系统提供服务，是整套系统的核心。  
linktv：web前端部分，目前仅适应PC端的chrome系列浏览器。
